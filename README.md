# Regional Sales Performance Tracker

An enterprise-grade Power BI dashboard designed to track, analyze, and visualize Fast-Moving Consumer Goods (FMCG) sales performance. This project transforms raw transaction data into actionable business insights regarding regional distribution, brand dominance, and chronological sales trends.

## 🚀 Key Features & Insights
* **High-Level KPIs:** Monitored over 298.10M in Total Sales Value, 676.32 Tons in volume, and an average price of 440.77K per ton.
* **Brand Dominance Mapping:** Visualizes market share where the flagship brand (*DIPLOMA*) commands over 60% of total revenue.
* **Territory Analysis:** Granular performance breakdowns across specific operational territories (e.g., Bahaddarhat, Banasree, Feni, Cumilla).
* **Time-Series Forecasting & Trends:** Tracks daily sales fluctuations over a 1-month period (December), highlighting recurring weekly cyclical patterns and a massive end-of-month sales surge driven by primary categories like *MILK*.
* **Dynamic Slicing:** Fully interactive filtering by Area, Category, Brand, and Territory.

## 🛠️ Tech Stack
* **BI Tool:** Microsoft Power BI Desktop
* **Data Source:** Relational Sales Databases / CSV
* **Data Modeling:** DAX (Data Analysis Expressions) & Power Query
